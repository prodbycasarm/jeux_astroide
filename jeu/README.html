Galactis

Galactis est un jeu de survie dans l'espace. Évitez les obstacles et survivez le plus longtemps possible !

Contrôles

  • W : Haut
  • S : Bas
  • A : Gauche
  • D : Droite

Comment jouer

  1. Cliquez sur le bouton Commencer pour démarrer la partie.
  2. Déplacez votre vaisseau avec les touches W, A, S, D pour éviter les astéroïdes.
  3. Votre score s'affiche en haut de l'écran.
  4. Vous avez 3 vies au début. Le jeu se termine lorsque vous n'avez plus de vies.
  5. Cliquez sur Sortir du jeu pour quitter.

Fichiers du projet

  • mon_jeu.html : Fichier principal du jeu
  • style.css : Styles du jeu
  • script.js : Logique du jeu
